This website is a community project to develop tracing guides for remote volunteers to add open geographic data to OpenStreetMap. Guides are an ordered collection of discrete mapping tips. Anyone can contribute!

If you'd like to contribute a tip or guide (we suggest starting with a tip), please submit a Pull Request to the community repo on GitHub at https://github.com/hotosm/tracing-guides. You can view previous pull requests at https://github.com/hotosm/tracing-guides/pulls.
